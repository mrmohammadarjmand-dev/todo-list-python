## 🎉 Version 2.0 — Modern GUI Release

Version 2.0 of the To-Do List App is now available!  
The application has been fully redesigned with a sleek graphical interface while keeping the lightweight, JSON-based storage.

**What's new in v2.0:**

- 🎨 **Beautiful modern UI** – custom color palette, flat design, and styled `ttk` buttons with hover effects
- ✓ **Visual task status** – done tasks are marked with a "✓", pending tasks with "○"
- 📊 **Live task counter** – shows how many tasks are still remaining
- ⌨️ **Enter key support** – quickly add a task by pressing Enter
- 🖥️ **Auto-centered window** – the app opens in the center of your screen
- 💾 **Automatic JSON saving** – all changes are saved instantly, just like before

**Existing features (v1.x) still included:**
- Add, delete, and mark tasks as done
- Clear all tasks with a confirmation prompt
- Scrollable task list
- No external dependencies – uses only Python's built-in `tkinter` and `json`

The source code is free, open-source, and perfect for learning Python GUI development or as a personal productivity tool.
